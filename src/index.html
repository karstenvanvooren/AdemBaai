Beweeg je handen — de golf reageert in hoogte, kleur en muziek (piano links, viool rechts).

Zonder camera kabbelt het rustig; met camera wordt het een instrument. Geluid start automatisch na je klik.

Beweeg Baai — Info

Een schermvullende “baai” die jouw beweging vertaalt naar een horizontale golf. Links kies je pianoklanken, rechts viool. Hoogte (Y) bepaalt toonhoogte en helderheid; intensiteit bepaalt dynamiek. Alles draait lokaal in je browser.

Code & instructies: je eigen GitHub repo.
